# 《记忆的缝隙》医学内容与敏感性审核台账

> 版本：v0.1  
> 最近来源核验：2026-06-22  
> 状态：等待专业审核  
> 适用内容：GDD、叙事脚本、科普页、宣传文案、测试问卷

## 1. 审核目的

本台账用于防止游戏在迭代过程中：

- 把叙事谜题描述为认知筛查、诊断或治疗；
- 把阿尔茨海默病表现为正常老化、个人失败或“爱能治愈”的疾病；
- 暗示所有患者会以同样速度、顺序或方式变化；
- 污名化、幼儿化或恐怖化患者；
- 把照护者的疲惫描述为不够爱；
- 提供超出一般科普范围的医疗判断。

本文不代替医生、专业机构或患者群体的审核。

“痴呆”可在正式政策名称、医学说明和原文引用中使用，但日常叙事优先以人为先。

## 4. 已采用的事实边界

### CLAIM-01

游戏表述：阿尔茨海默病会影响记忆、思考和日常活动，并可能随时间进展。

- 来源：SRC-WHO-2025
- 使用位置：GDD 项目概述、宣传简介
- 风险：不得推断每个人都会按游戏中的 D0–D4 顺序进展
- 状态：来源支持，等待专业审核

### CLAIM-02

游戏表述：值得关注的变化可包括忘记近期事件、放错物品、熟悉地点迷路、时间地点混淆、完成熟悉任务困难、对话或找词困难、视觉空间判断变化，以及情绪或行为变化。

- 来源：SRC-WHO-2025
- 使用位置：NARRATIVE_SCRIPT.md 的 guide.signs.body
- 风险：这些变化可能有多种原因，不能凭单项自行诊断
- 状态：来源支持，等待专业审核

### CLAIM-03

游戏表述：出现持续变化并影响生活时，应记录具体情况并寻求正规医疗机构专业评估。

- 来源：SRC-CN-ACTION、SRC-WHO-2025
- 使用位置：科普页免责声明与行动建议
- 风险：不得指定未经审核的量表、科室或治疗方案
- 状态：来源支持，等待地区化审核

### CLAIM-04

游戏表述：保持熟悉活动和社会参与、改善日常环境、寻求家人与专业支持，可以帮助维护生活质量与安全。

- 来源：SRC-WHO-2025
- 使用位置：科普页行动建议
- 风险：不得承诺阻止或逆转疾病
- 状态：来源支持，等待专业审核

### CLAIM-05

游戏表述：照护者需要休息，并可向家人、社区和专业人员求助。

- 来源：SRC-WHO-2025
- 使用位置：林秀兰人物设定、科普页
- 风险：不得把求助描述为抛弃或不够爱
- 状态：来源支持，等待敏感性审核

## 5. 游戏机制声明

以下文案必须出现在科普页首屏或“关于游戏中的谜题”区域：

> 照片排序、数字连接、颜色和形状辨识只用于叙事体验，不是医学筛查，也不能产生任何认知健康结论。

禁止：

- 根据解谜时间、错误次数或提示使用情况生成分数；
- 将游玩数据命名为记忆力、认知力、风险等级或脑龄；
- 通过“你表现很好，所以不必担心”等方式给出安慰性诊断；
- 暗示某类玩法可以治疗或延缓阿尔茨海默病；
- 将测试参与者的游玩记录和健康状态关联。

## 6. 叙事敏感性检查

### 6.1 许志远

- 他有职业经历、幽默、偏好和主动选择，不只由疾病定义；
- 他会接受提示，但不是被玩家或林秀兰支配；
- 忘记姓名不等于忘记所有感受或失去人格；
- 方向错位是主观叙事机制，不声称是所有患者的临床体验。

### 6.2 林秀兰

- 她不以考问方式强迫回忆；
- 她先等待，再提供帮助；
- 她可以疲惫并寻求支持；
- 她不是“用爱治病”的工具人物。

### 6.3 结尾

必须保留：

- 许志远没有恢复林秀兰姓名；
- 林秀兰再次介绍自己；
- 许志远主动牵手；
- 主题落在关系与尊严，不落在奇迹恢复。

不得增加：

- 记忆突然完整闪回；
- “我全想起来了”；
- 医生宣布病情因训练好转；
- 把牵手解释为治疗效果。

## 7. 宣传文案红线

禁止使用：

- “最真实还原阿尔茨海默病患者的大脑”；
- “体验一次就能懂所有患者”；
- “测试你的记忆是否健康”；
- “用游戏预防/治疗阿尔茨海默病”；
- “爱可以战胜疾病”；
- “患者最终会忘掉一切”。

推荐使用：

- “从一位虚构老人的主观视角，体验熟悉信息逐渐变得不可靠”；
- “作品关注认知障碍患者的尊严、家庭沟通和陪伴”；
- “游戏不是医学筛查或诊断工具”；
- “不同患者的经历和需要并不相同”。

## 8. 专业审核流程

### 8.1 审核材料

向专业审核者提供：

1. GDD 第 2、3、5、10、16、17 节；
2. NARRATIVE_SCRIPT.md 全文；
3. 科普页可操作构建；
4. 主题理解测试问卷；
5. 本台账的事实边界和待确认项。

### 8.2 审核问题

- 疾病名称、表现与进展表述是否准确？
- 是否把个体化体验误写成共同临床事实？
- 科普页是否清楚区分一般信息和专业评估？
- 陪伴和环境建议是否安全、不过度承诺？
- 是否遗漏需要紧急或地区化处理的提示？
- 是否存在污名化、幼儿化或照护者道德绑架？
- 宣传文案是否可能让公众误解为诊断或治疗产品？

### 8.3 结论等级

| 等级 | 含义 | 处理 |
| --- | --- | --- |
| 阻塞 | 可能造成医疗误导、伤害或严重污名 | 发布前必须修改并复审 |
| 高 | 明显不准确或容易产生错误理解 | RC 前必须关闭 |
| 中 | 语气、代表性或可理解性问题 | 发布前处理或书面接受 |
| 低 | 润色与补充建议 | 记录后排期 |
